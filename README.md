# Gorilla Cosmetic Notifier
Gorilla cosmetic notifier is a mod that notifies you when a *special gorilla* joins the server, unlike normal notifiers/identifiers that tell you what cosmetics a monke owns *even without having them equipped.*

**Cosmetic List:**
*AAC Badge*
*Finger Painter badge*
*Illustrator badge*
*Forest Guide Stick*
*Developer Stick*
*Admin Badge*
